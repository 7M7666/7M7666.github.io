计算机二级/c语言2

细碎的知识点

c程序

源文件后缀名为.c 编译后为.obj的目标文件 把目标文件和各种库函数链接在一起后变成.exe可执行文件。

存储形式

8bit=1byte b->kb->mb…中间关系为x1024 数据存放位置为其地址

标识符

是标识名字的有效字符序列，可以理解为C程序中的单词。
标识符的命名规则是；
（1）标识符只能由字母、效字和下划线组成，字母区分大小写。
（2）标识符的第一个字符必须是字母或下划线，不能为数字。
c语言标识符分如下3类
（1）关键字。它们在程序中有固定的含义，不能另作他用。如 int、for、switch 等。
（2>预定义标识符。预先定义并具有特定含义的标识符。如define、include 等。
（3）用户标识符。用户根据帮要定义的标识符，符合命名规则且不与关键字相同。

常量和变量

常量是指在程序运行过程中，其值不能改变的量。整型常量、实型常量、字符常量、字符串常量、符号常量5种。在运行过程中其值可以改变的量称为变量：C语言中没有字符串变量：存放字符串使用字符数组。

整型数据

整型数据有三种表示形式：十进制、八进制和十六进制，但没有二进制形式。八进制整型常量以数字0开头，十六进制整型常量以”0X”或”0x”开头。在八进制中，不会出现数字8。整型变量包括基本整型（int）、短整型（short）、长整型（long）和无符号整型（unsigned）。基本整型占用4个字节，其具体占用字节数和取值范围请参考教材第9页。

实型数据

实型数据有两种表示形式：小数形式和指数形式。判定指数形式的合法性时，应注意E前后必须有数字，且E后必须是整数。实型数据分为单精度型（float）和双精度型（double），其中单精度型占用4个字节。

算术运算

算术运算符共有五个：加（+）、减（-）、乘（*）、除（/）和求余（%）。求余运算符要求运算对象为整型，而除法运算符要求运算对象也为整型。

运算符、表达式、语句

运算符与运算对象结合形成表达式，而表达式加上分号则构成表达式语句。运算对象可以是表达式、变量或常量。例如，赋值运算符与运算对象结合形成赋值表达式，赋值表达式加上分号又可以构成赋值语句。

运算符的优先级和结合顺序

运算符根据参与运算的对象数量可分为单目运算符、双目运算符和三目运算符。初学者应首先掌握运算符的优先级别，其次是单目运算符、算术运算符、关系运算符、逻辑运算符（除逻辑非！）、条件运算符、赋值运算符、逗号运算符。位运算符的优先级位于算术运算符与逻辑运算符之间。大多数运算符的结合顺序为自左向右，但单目运算符、条件运算符和赋值运算符的结合顺序为自右向左。

printf 函数

printf 函数的格式为：printf(输出控制, 输出列表)。输出控制是一对双引号括起来的字符串，包含格式说明和原样信息。

printf 函数中的格式说明

在 printf 函数中，%d 对应整型，%f 对应单精度实型，%c 对应字符型，%o 对应八进制无符号整型，%x 对应无符号十六进制整型，%u 对应无符号整型，%e 对应指数型，%s 对应字符串型。可以在 % 和格式字符之间加上一个数字来控制数据所占的宽度和小数位数。

scanf 函数

scanf 函数用于输入数据，其输入项需要带取热站符 &。当使用键盘输入多个数据时，数据之间用分隔符隔开。分隔符包括空格符、制表符和回车符，但不包括逗号。

结构体类型的说明

结构体是多个不同类型数据的集合，其类型说明格式为：struct 结构体名 { 类型1 成员名1; 类型2 成员名2; ... }。整个结构体定义构成一个数据类型，与整型 int 具有相同地位。可以使用 typedef 将结构体类型替换成一个简短的标识符。

结构体变量的定义

结构体变量是用说明的结构体类型所定义的一个变量，与结构体类型本身是不同的。一个结构体变量所占字节数为所有成员所占字节数之和。例如，定义两个结构体变量 a，每个变量占4个字节。

结构体成员的引用

引用结构体成员有三种方式：（1）结构体变量名.成员名；（2）指针变量名->成员名；（3）(*指针变量名).成员名。. 称为成员运算符，-> 称为结构指针运算符。

链表

链表是由多个节点构成的，每个节点是一个结构体变量。节点可以分为数据域和指针域，数据域用来存放数据，指针域用来指向下一个节点。链表是C语言和计算机基础中的难点，需要深入理解掌握。

共用体

共用体的使用格式与结构体相似，定义关键字为 union。共用体所占字节数为所有成员中字节数最大的那个。

文件类型指针

文件指针是一个指向结构体类型的指针，定义格式为：FILE* 指针变量名。在使用文件时，都需要先定义文件指针。

文本文件与二进制文件

文本文件存放的是字符的ASCII码，二进制文件存放的是数据的二进制形式。例如，“100”在文本文件中是存储为三个字符的ASCII码，而在二进制文件中则是存储为二进制形式。

打开文件

文件的打开方式如下：FILE *fp = fopen("cWVab.c", "rb")。其中，rb 表示以只读方式打开一个二进制文件。其他打开方式详见教材第127页。

文件函数

  • feof() 函数用于判断文件结束，调用形式为：feof(FILE *fp)。如果文件结束，则函数返回1，否则返回0。
  • fseek() 函数用于设置文件的位置，调用形式为：fseek(文件指针, 位移量, 移动起始点)。
  • FILE* fopen("C:\uxiaoyu.txt", "r"); 打开一个文本文件。
  • fseek(fp, 12, SEEK_SET); 表示将文件指针的位置移动到文件起始位置后面的第12个字节的位置上。
  • fseek(fp, -12, SEEK_END); 表示将文件指针的位置移动到文件末尾位置前面的第12个字节的位置上。

函数的声明

函数要“先定义后调用”，或“先声明再调用后定义”。函数的声明一定要有函数名、函数返回值类型、函数参数类型，但不一定要有形参的名称。

函数的参数及值传递

形式参数简称形参，是定义函数时函数名后面括号中的参数；实在参数简称实参，是调用函数时函数名后面括号中的参数。实参和形参分别占据不同的存储单元。实参向形参单向传递数值。不是复制！！

要求掌握的库函数

sqrt()计算平方根函数，fabs()绝对值函数，pow()幂函数，sin()正弦函数等。

指针变量

指针变量的定义
指针变量是用来存储地址的，而一般变量是存储数值的。指针变量可指向任意一种数据类型，但不管它指向的数据占用多少字节，一个指针变量占用四个字节。

指针变量的初始化

格式为：类型名 *指针变量名；二维指针int **p;可以理解为基类型为(int *)的类型。

指针变量的引用

& 是取地址符，* 是间接访问运算符，它们是互逆的两个运算符。在指针变量名前加间接访问运算符就等于它所指向的量。

指针的运算

*p++ 和 *(p++) 之间的区别：p++ 是地址变化，(p++) 是指针变量所指的数据变化。一个指针变量加一个整数不是简单的数学相加，而是连续移动若干地址。当两个指针指向同一数组时，它们可以比较大小进行减法运算。

算法的基本概念

算法是一组有穷的指令序列，是解题方案的准确而完整的描述。算法不等同于程序，也不等于计算方法。算法的基本特征包括：

  • 确定性：算法中每一步都必须有明确定义，不允许有多义性。
  • 有穷性：算法必须在有限的时间内完成，即在有限步骤后终止。
  • 可行性：算法原则上能够精确地执行。
  • 拥有足够的情报：算法必须具备解决问题的足够信息。
    算法由数据对象的运算和操作以及其控制结构两部分组成。基本运算和操作包括算术运算、逻辑运算、关系运算、数据传输。基本控制结构有顺序、选择、循环。算法设计方法包括列举法、归纳法、递推、归纳、减半递推技术等。

算法的复杂度

算法复杂度是衡量算法效率的度量，包括时间复杂度和空间复杂度：

  • 时间复杂度：指执行算法所需计算工作量。包括编译时间和运行时间。
  • 空间复杂度：指执行算法所需内存空间，包括算法程序所占空间、输入数据的初始空间、算法执行过程中额外空间的需求。
    时间复杂度和空间复杂度并不相同。

数据结构的基本概念

数据是客观事物的符号表示，是能输入到计算机中并被计算程序识别和处理的符号的总称。数据元素是数据的基本单位。数据对象是性质相同的元素集合。数据结构是由数据对象中所有数据成员之间的关系组成的结构。

逻辑结构和存储结构

数据结构可分为逻辑结构和存储结构：

  • 逻辑结构：描述数据元素之间的逻辑关系，与数据的存储无关，面向问题的，独立于计算机。
  • 存储结构：又称物理结构，指数据在计算机中的存放方式，面向计算机，包括数据元素的存储方式和关系的存储方式。
    逻辑结构和存储结构不一定一一对应。常见的存储结构有顺序、链表、索引等，不同的存储结构对数据处理的效率不同。

线性结构和非线性结构

线性结构的特点是：

  • 有一个根节点。
  • 每个节点最多有一个前驱，最多有一个后继。
    非线性结构不满足线性结构的特点。常见的线性结构有栈、队列、双向链表，而非线性结构有树、二叉树。

线性表及其顺序存储结构

线性表是由一组数据元素构成，数据元素的排列只取决于自己的序号，元素之间存在线性关系。在复杂线性表中，由多个数据元素组成的单个数据元素称为记录；由多个记录构成的线性表称为文件。
